# Test Framework to test SpringBoot Application using Rest Assured and JUnit4

This framework consists of:
````
SpringBoot - 3.2.3
Java 17
Maven – 3.9.6
JUnit - 4.13.2
Rest Assured - 5.3.2
Maven Compiler Plugin – 3.12.1
Maven Surefire Plugin – 3.2.3
````

To run the tests, use the command
````
mvn clean test
````
